Не пытайтесь запустить бота через main.py, он задеплоен на pythonanywhere.
Бот был написан на версии Python 3.9 и стабильно работает на этой версии. Используемые библиотеки: pydub, numpy, soundfile, scipy, python-telegram-bot 
Проект представляет собой телеграмм бота, в котором можно редактировать звуковые файлы. Представлены следующие методы редактирования:
1. Басс буст
2. Замедление или ускорение музыки
3. Гроссбит
4. Шумоподавление
5. Обрезка аудиофайла
6. Разделение голоса и звука
7. Ревёрб
8. Ревёрс
Для того что бы воспользоваться функционалом бота пользователю нужно зарегистрироваться (если у пользователя нет аккаунта) и войти в аккаунт. 
Для регистрации нужно придумать логин и пароль. Регистрация не представляется возможной если:
- Пользователь с таким ником уже существует
- Пользователь не ввёл логин или пароль
В меню помощи представленна информация как использовать бота, а так же информация о проекте.
В меню редактирования представлен выбор методов редактирования. После выбора метода бот просит пользователя прислать файл и опционально параметр изменения. После того как бот изменил файл он будет отправлен пользователю
